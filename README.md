# SD Prompt Manager Ultimate

Created by **Nuruneko**

Stable Diffusionなどの画像生成AIで使用するプロンプト（呪文）を、直感的に作成・管理・保存できるオールインワンツールです。
PCの大画面でも、スマホのタッチ操作でも快適に動作します。

## ✨ 主な機能
* **直感的な操作:** タグをクリックして追加、ドラッグ＆ドロップで並び替え。
* **スマホ対応 (PWA):** スマホのホーム画面に追加して、アプリとして使用可能。
* **画像解析 (PNG Info):** 生成画像をドロップすると、プロンプト情報を自動で読み取ります。
* **編集モード:** カテゴリーやグループを自由に並び替え・リネーム可能。
* **翻訳機能:** 日本語 ⇔ 英語 の自動翻訳を搭載。
* **データ保存:** ブラウザに自動保存＋JSONファイルでのバックアップ/復元に対応。
* **プライバシー:** 入力したデータはすべてあなたの端末内にのみ保存され、外部に送信されることはありません。

## 📱 使い方
1. **[Web版を開く](https://nuruneko3710.github.io/sd-prompt-manager-ultimate/)**
2. **スマホの場合:** ブラウザのメニューから「ホーム画面に追加」を選択。
3. **PCの場合:** ブックマークするか、そのまま使用。
